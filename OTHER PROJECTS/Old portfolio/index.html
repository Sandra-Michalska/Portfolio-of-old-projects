Who I am

My name is Sandra Michalska. I've always thought that creating web sites is something extremely involving. It already made my heart beat faster about 10 years ago (and still does!). Last year I felt that it's high time to take it seriously and start to learn coding for real!

It all started with reading about HTML, CSS and JavaScript on the tram or at home after work, while over the last few months it's been almost like a full-time job for me. Even though there is still so much interesing stuff for me to master, I believe now is the right time to take it to the next level and get my first job as a Front-End Developer!

How I learn to code

This is what I've managed to do so far:

  • I'm finishing the Complete Web Developer Course on Udemy, as well as doing the Free Code Camp course. The Udemy course features a number of web development related topics such as: HTML, CSS, JavaScript, jQuery, Twitter Bootstrap, Wordpress, PHP, MySQL, APIs and mobile apps. I pay particular attention to the frontend material but going through the PHP or MySQL parts gives me some basic understanding of the backend issues, too.
  • I am reading various books dealing with HTML, CSS, Java Script, jQuery and Responsive Web Design. As most of the books come with hundreds of short exercises, I've already had a lot of practice.
  • I also look for answers and inspiration on the Internet. Apart from using W3Schools, I read different forums posts, go through tutorials or try to figure out solutions to various coding issues myself.
  • In order to use the knowledge even more, I work on different projects (see below). I also decided to create this portfolio. As my experience is growing from day to day, there are more advanced web sites to follow soon for sure;)

The skills I'm developing

Currently I'm also learning Angular.js, LESS and Git. When it comes to my future plans, Photoshop is the next on my list. I also have some experience with Corel Draw 12.

Once I'm done with learning all of the front-end development basics, I'm planning to focus on and master CSS and JavaScript.

My expectations

What I'd like to get most from a job as a Front-End Developer is as follows:

  • A possibility to develop my skills, both in the technologies I'm learning now and in new ones,
  • Friendly atmosphere and assistance from other co-workers to help me learn best coding practices,
  • And... The above two are what matters most to me!
